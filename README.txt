263-SONLI MAKTAB VEB-SAYTI
==========================

Loyiha: Next.js 14 + TypeScript + Tailwind CSS + Framer Motion + next-intl

ISHLATISH:
----------

1. Loyiha papkasiga kiring:
   cd maktab-263

2. Kutubxonalarni o'rnating:
   npm install

3. Ishlab chiqish rejimida ishga tushiring:
   npm run dev

4. Brauzerda oching:
   http://localhost:3000/uz   — O'zbek tili
   http://localhost:3000/ru   — Rus tili

PRODUCTION BUILD:
-----------------
npm run build
npm start

TEXNOLOGIYALAR:
---------------
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animatsiyalar)
- next-intl (UZ/RU til almashtirish)
- Lucide React (iconlar)

ESLATMA:
--------
Rasmlar hozircha Unsplash placeholder sifatida ishlatilgan.
Keyinroq maktabning haqiqiy rasmlari bilan almashtirishingiz mumkin.
